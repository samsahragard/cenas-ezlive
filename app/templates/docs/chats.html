Chat handoff folder

Per-session handoff logs from each AI agent (aick, ck, samai) working on the codebase. New chats get archived here at the end of each session so the next agent — or future-me picking up cold — has the full picture without having to dig through git history.

To add a new chat: drop app/templates/docs/<slug>.html and append one line to CHAT_PAGES in app/web/developer_chat.py. The slug becomes the URL (/partner/developer/app/<slug>) and the route resolves it automatically — no other touch needed.
