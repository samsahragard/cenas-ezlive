2. Authentication tiers

Three nested gates. A keypad session implicitly satisfies Tier 1 + 2 (the keypad-login handler sets auth_ok and — for partner role — partner_auth_ok at the same time). A driver session is its own track, gated by session['driver_id'].

Tier 1 — Site password (EZLIVE_PASSWORD)

  • Default value: cenas (env var, can rotate on Render).
  • Sets session['auth_ok'].
  • Enforced by auth._gate as a Flask before_request hook.
  • Exempt: /install, /login, webhook endpoints, ingest endpoints, /healthz, /privacy.

Tier 2 — Partner password (PARTNER_PASSWORD)

  • Default value: Cenas7234.
  • Sets session['partner_auth_ok'].
  • Enforced by store_bp.before_request for any URL under /partner/*, plus a belt-and-suspenders _enforce_partner() on each Developer-Chat / App-Docs / Partner-Team route.
  • Unlocks: Developer Chat, App Docs, Partner-only orders / analytics / team admin.

Tier 3 — Per-person 5-digit passcode (keypad)

  • Stored on users.passcode_hash (werkzeug hash).
  • Sets session['user_id'] → g.current_user on every request via keypad_auth._attach_current_user.
  • Role from users.permission_level: partner > corporate > gm > manager > expo > corporate-driver.
  • Forced passcode change on first login (first_login_done = False).
  • Side track: driver session (session['driver_id']) lives in parallel — Driver model is separate from User. Driver login at /driver/login, same 5-digit-PIN flow on drivers.passcode_hash.

11. Background workers + scheduled work

JobLivesHow it runs
Produce IMAP pollerapp/services/produce_ingest.py Daemon thread spawned by create_app(). Gated on PRODUCE_INGEST_ENABLED=1. fcntl file lock at /var/data/produce/.ingest.lock so only one gunicorn worker polls. 60s interval. Parses Alvarado / J. Luna price sheets from orders@cenaskitchen.com.
Idempotent column backfillapp/__init__.py Runs on every boot inside create_app(). Inspects DDL with sqlalchemy.inspect and ALTERs in any missing columns from migrations 8 / 10 / 11 / 12 / 13 / 15. Self-healing pattern that replaces alembic Pre-Deploy.
Order total backfillapp/__init__.py One-shot per boot — computes orders.total_amount for any rows where it's NULL but items are present. Capped 500 / boot.
Sam-as-partner seedapp/__init__.py If users table is empty, inserts Sam with bootstrap passcode 12345, first_login_done = False.
ezCater known-driver seedapp/__init__.py Idempotent seed of 36 drivers from Sam's 5/10 screenshots into ezcater_known_driver.
