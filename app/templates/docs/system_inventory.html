System Inventory — Cenas Kitchen

Author: ck. Initial version 2026-05-15.
Purpose: a single operational MAP of every system Cenas Kitchen depends on, every machine that runs something, and where each credential lives — without recording any actual secret values.

NON-NEGOTIABLE RULE. This doc records only the location of credentials (env var names, file paths, 1Password references). Never paste an actual key, password, token, PIN, or any value that looks like a secret. Aggregating real secrets into one document creates a catastrophic single point of failure — the opposite of what this inventory is for. If you find yourself about to paste a value, stop and record the location instead.
