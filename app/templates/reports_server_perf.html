Notes

  • Avg drink = order open → first drink rung in.
  • Avg apps (n) = order open → first appetizer; n = tickets that had an appetizer.
  • Avg entree = order open → first entree rung in.
  • Drink→Entree gap = avg time between first drink and first entree on same check. Bigger = better service (server made two trips). Small (under ~1 min) = rushed (server took whole order at once).
  • Avg duration = order open → check closed.
  • CC tabs = sum of payment.amount for CREDIT payments (food + drinks + tax, before tip).
  • CC tips = sum of payment.tipAmount for CREDIT payments (added on top of the tab).
  • Tip % = total CC tips / total CC tabs × 100. Cash sales excluded — Toast doesn't track cash tips.
  • Voided orders + voided checks + voided selections excluded.
  • Only employees with Server / Server Trainee / Bartender / Host roles included. Managers/cooks/owners excluded.
