# Cenas Kitchen Driver (mobile shell)

Capacitor-based native wrapper for the Driver Portal at
<https://app.cenaskitchen.com/driver/login>.

## What this is for

The web-only Driver Portal stops tracking GPS the moment Chrome is
backgrounded / closed / the phone goes in a pocket. This mobile shell
wraps the same web pages in a native iOS + Android app, with
`@capacitor-community/background-geolocation` running as a native
foreground service that POSTs to `/driver/track` regardless of what
the WebView is doing. That gives us real background tracking that
survives Chrome closed, app-switched, screen off.

## Layout

```
mobile/
├── capacitor.config.json    # Capacitor config (appId, server.url, plugins)
├── package.json             # Capacitor + plugin deps
├── www/index.html           # Fallback HTML (server.url normally takes over)
└── android/                 # Generated by `npx cap add android` (in CI)
└── ios/                     # Generated by `npx cap add ios` (Phase B, needs macOS)
```

The `android/` and `ios/` platform folders are NOT committed — they're
generated fresh in CI (or locally) by `npx cap add <platform>` from the
`package.json` deps + `capacitor.config.json`.

## Building (CI, recommended)

GitHub Actions workflow `.github/workflows/mobile-android.yml` builds
a debug APK on every push touching this folder and uploads it as a
workflow artifact. Pull the artifact, sideload on a test device.

## Building (local Android, optional)

Requires: Node.js 18+, Java JDK 17, Android Studio SDK.

```bash
cd mobile/
npm install
npx cap add android
npx cap sync android
cd android && ./gradlew assembleDebug
# APK output: app/build/outputs/apk/debug/app-debug.apk
```

## iOS (Phase B)

Same project. Add the iOS platform on a Mac:

```bash
cd mobile/
npx cap add ios
npx cap sync ios
npx cap open ios  # opens Xcode, then Product → Archive
```

Requires:
- macOS + Xcode 15+
- Apple Developer Program enrollment ($99/yr)
- App Store Connect listing

Or use a CI service (Codemagic / Ionic Appflow) for iOS builds
without owning a Mac.
